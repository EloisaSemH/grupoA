O que é Git?

O Git é um software livre (GNU General Public License) inicialmente foi projetado e desenvolvido por Linus Torvalds, para o desenvolvimento do Kernel Linux, é um dos vários Sistemas de Controle de Versão de arquivos. Através dele é possível desenvolver projetos na qual diversas pessoas podem contribuir simultaneamente no mesmo, editando e criando novos arquivos e permitindo que os mesmos possam existir sem o risco de suas alterações serem sobrescritas. Umas das principais aplicações do Git é justamente esta, permitir que um arquivo possa ser editado ao mesmo tempo por pessoas diferentes.

Como explicado no site git-scm, o desenvolvimento da plataforma Git se deu a necessidade de desenvolver o próprio Sistema Operacional Linux , no início (1991-2002) as mudanças desse projeto de software de código aberto eram repassadas com patches e arquivos compactados, a partir de 2002 esse projeto começou a usar o BitKeeper, um sistema DVCS proprietário.

Em 2005, o relacionamento entre a comunidade que desenvolvia o kernel e a empresa que comercializada o BitKeeper se desfez pois a licensa gratuita foi revogada. Em contra resposta, a comunidade desenvolveu sua própria ferramenta de controle de versão. Foi assim que em 2005 foi desenvolvido o Git.
