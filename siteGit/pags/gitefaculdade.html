Como um grupo de faculdade pode usar Git para colaborar em um projeto?

Diferente do que muitos pensam, o Git não se encaixa unicamente a projetos grandes de empresas renomadas, também cabem nele projetos de empresas pequenas, e principalmente, de projetos academicos

Mas em que o Git me ajudaria?

Simples, o git é uma ótima ferramenta para criação de projetos e armazenamento de fontes, ou seja, uma vez que um projeto está sendo desenvolvido, ele pode ser armazenado em uma dos diversos softwares que trabalham a partir dele, assim, caso futuramente voce se esqueça, ou precise revisar o código feito anos atrás, voce o terá a sua disposição

Mas essa é a uica vantagem?

Não jovem gafanhoto. O git também é uma ótima ferramenta usada na programação de fontes em que exite mais de um programador pr trás das linhas de código, sendo assim, ao invés de salvar e compartilhar sua criação por meio de um Pen Drive (que pode muito facilmente se perder, deixando aassim suas informações em um constante estado de vulnerabilidade).

Ao armazenar em um software Git (GitHub, GitLab, Bitbucket, etc.) suas linhas de programação, voce poderá contar com a vantagem de programar com a presença de um Controle de versões, ou seja, a cada modificação realizada, serão armazenadas informações como:

  • Nome de quem modificou;
  • Data da modificação;
  • Modificações realizadas;

A partir dessas informações voce poderá decidir se aceita ou rejeitas as modificações realizadas.
