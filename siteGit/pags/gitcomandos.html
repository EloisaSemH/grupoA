Os principais comando do Git

git init - cria um nono repositório git clone /"caminho para o repositório"/ - cria uma cópia de trabalho em um repositório local git pull - atualiza o repositório local git merge (branch) - mescla outro branch ao seu git add (arquivo) - adiciona mudanças aos index git commit -m "comentários das alterações" - grava as alterações no repositório local git push origin (nome do branch) - envia alterações ao repositório remoto git remote add origin - conecta em um servidor git checkout -- (arquivo) - subscreve as alterações locais

O que é git pull?

Incorpora alterações de um repositório remoto num branch atual.

O que são branches?

Branches são cópias de alguma parte do projeto, para que você possa trabalhá-las separadamente.
